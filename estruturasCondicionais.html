Estruturas Condicionais

IF

A estrutura "se" (IF) é um comando que permite que um programa tome uma decisão ao executar um bloco de código apenas se uma determinada condição for verdadeira. Quando essa condição não é atendida (é falsa), o programa segue o seu fluxo normal após o bloco condicional, sem executar as instruções dentro do "se".

Switch-case

A estrutura "escolha caso" (ou switch-case) é uma estrutura condicional de seleção múltipla que permite que um programa execute diferentes blocos de código com base no valor de uma única variável.
