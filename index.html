Projeto Final Front-End

FRONT-END
A Trindade do Front-End é composta por Html, Css e JavaScript.
HTML(HyperText Markup Language) - É a linguagem de marcação padrão para criar e estruturar páginas da web. Ela utiliza um sistema de "tags" para definir elementos como títulos, parágrafos, imagens e links, que são interpretados pelos navegadores para apresentar o conteúdo de um site.
CSS(Cascading Style Sheets) - É uma linguagem de estilo utilizada para descrever a aparência de documentos escritos em HTML ou XML. Ela define o estilo visual de um site, controlando elementos como fontes, cores, espaçamento, layout e alinhamento.
JavaScript(JS) - É uma linguagem de programação usada para adicionar interatividade e conteúdo dinâmico a páginas da web, tornando-as mais do que apenas conteúdo estático, e para criar aplicações complexas em geral, incluindo o desenvolvimento web do lado do cliente e do lado do servidor. Ele funciona em conjunto com HTML (estrutura) e CSS (estilo) para dar vida aos sites, permitindo coisas como menus suspensos, galerias de fotos animadas, mapas interativos e atualizações de conteúdo em tempo real.
